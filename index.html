About

To make things clear, this is not an official Cypress Bay website. This is a website developed by students to help other students. If you would like to visit the actual website please navigate to this page.

This project is dedicated to delivering clear and reliable news to the students. One problem from previous school year is the uncertainty of information. Although Cypress has an official website, it does not cover everything students would like to know. This website fixes that problem and delivers news and updates right to you. You can meet the community behind this project through the discord server.

Contributing

One man cannot complete this job with ease. Regarding website help, please visit the github repository and look at the issues tab. If you would like to help add news and updates please visit the contact page.

One thing I need help with right now is website design so if you are good at design or anything of that sort, you already know what to do. Of course, if you cannot help out in any of these fields, exposing this website is the best you can do to help out fellow peers.

Extra Information

I would love to put club, class, and other information down in a different tab. Having club information available would help new students in seeing all updates for each club and aiding the choice in picking which ones to join. Like I have mentioned earlier, I do not have a giant ear and cannot provide information for every single club and class known. If you would like to help out with your club or any updates you would like to share, please head to the contact tab and message me.
